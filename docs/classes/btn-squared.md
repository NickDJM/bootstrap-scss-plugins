# Rounded Buttons

Squares the corners of buttons to get a 90 degree angle.

## Default Classes

```scss
.btn-squared
```

## Usage

### Styles

```scss
@import 'path/to/node_modules/bootstrap-scss-plugins/scss/button-squared';
```

### HTML

```html
<button class="btn btn-squared btn-primary">Click me!</button>
```
